js provides static method that belongs to class instead of an nstance of an that class. these methods are called directly on class itself

points to remember

  • the static keyword is used to declare a sttaic methos
  • the static method methos can be any name
  • class contain more than one static method also
  • whwn we declare the same name of methods multiple times it will take the last one
  • we can use the this keyword to call the static methos into another staic methos
  • but we cant call the static method inside the another sttaic method by using this keyword
  • in such case we can call static eitehr using classname or property of constructor
