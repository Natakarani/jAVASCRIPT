promises

what is promise

promise is a inbuilt constructor of javscript

a promise object represents a operation which producted or will eventually prodiuce a value,. promises provide a robust way to execute the (posssibly pending) result of asynchronous result

states of promises

there are 4 states

  • pending
  • fulfilled
  • rejected
  • settled

pending state

the operation not yet completed or not rejected (i.e, not yet started) that state we can call it as pending sttae(.then() is going to handle the pending state)

fulfilled state

operation is successfully completed and promise is fulfilled with response (syntax is .then() for to handle the fulfiled state)

rejected state

the error will gets due may be any reason (line catch()) (syntax .catch() for to handle the rejected state)

settled sttae

promise said to be settled that may be rejected or successfull (that state we can call it as settled state). promise is immutable it wont goes to pending sttae again
